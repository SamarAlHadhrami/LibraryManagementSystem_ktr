Library Management System (Console Based)

This project is a simple Library Management System made using Python.
It works only in the terminal and does not use any GUI or web framework.

This project is created as part of a 30 Days Internship program.

Features
- Add a new book
- Show all books
- Issue a book
- Return a book
- Menu driven program

Concepts Used
- Python Classes and Objects
- Constructor (__init__)
- OOPS concepts
- Exception handling
- Custom exceptions
- List data structure

How to Run the Program
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal or command prompt
4. Go to the project folder
5. Run the program using:
   python library_management.py

Rules Followed
- Console based program only
- No GUI or Tkinter used
- No web frameworks used
- Only Python (.py) file used
- Menu driven program
- Clean and readable code with comments
